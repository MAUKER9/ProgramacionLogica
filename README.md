# Programación Lógica

Proyecto académico desarrollado en **Java con JavaFX** para la materia de Programación Lógica.

## Contenido

- Ejemplos de programación **imperativa** y **funcional**.
- Implementación de **interfaces gráficas** en JavaFX.
- Ejercicios prácticos como:
  - Factorial (imperativo y funcional).
  - Formularios con validación básica.
  - Manejo de GUI con eventos.

## Requisitos

- [Java JDK 17 o superior](https://adoptium.net/)  
- [IntelliJ IDEA](https://www.jetbrains.com/idea/)  
- Git (para clonar el repositorio)
